## 使用 Go 编写的 web 应用

内容参考《Head First PHP & MYSQL》中的示例

除 MYSQL 驱动外，其余完全由 Go 内置的 package 编写，体现 Go 本身的强大和简洁

包含内容
- 登录
- session 验证
- 图片上传
- template
- 路由配置
- 数据库联表查询

使用的包
- log
- time
- bytes
- net/http
- database/sql
- html/template
- crypto/aes
- crypto/cipher
- crypto/sha1
- encoding/base64
- encoding/json
- encoding/hex
- image
- image/gif
- image/jpeg
- image/png
- io
- io/ioutil
- os
- strings
- strconv
